# github-pages
Criando site do portfólio
Desenvolvimento do portfólio de Bootcamp I na Entrega Intermediária
Esse repositório contém o desenvolvimento do meu site de portfólio pessoal, criado com HTML, CSS, JavaScript e Bootstrap, e publicado por meio do GitHub Pages.

O objetivo do projeto é apresentar minhas informações acadêmicas, habilidades, projetos, experiências e formas de contato de maneira organizada. O site foi personalizado a partir de um template Bootstrap e adaptado para apresentar minhas competências e meus interesses na área de tecnologia e desenvolvimento web.

Além do processo de desenvolvimento do site, o repositório também registra o uso de Git e GitHub para versionamento, organização e publicação do projeto.
